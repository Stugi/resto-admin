# Прототипы страниц

Сюда складывай HTML-прототипы (макеты) экранов.

## Как использовать

1. Сохрани `.html` файл с прототипом в эту папку
2. Назови по экрану: `dashboard.html`, `booking-form.html`, `settings.html`
3. При вёрстке скажи Claude: "Посмотри прототип `docs/prototypes/название.html`"

## Именование

-   `resto-admin-mobile` — версия для мобилки

Claude автоматически учитывает прототипы из этой папки как референсы.
